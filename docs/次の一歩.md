## 🎯 まず最初にやること（5分でできる）

### バディの性格を仕上げる
1. Dashboard (`http://localhost:18789`) を開く
2. 「**Identity**」 or 「**Soul**」セクションで:
   - 名前（呼ばれたい呼び方）
   - 一人称
   - トーン（フレンドリー / 落ち着いた / タメ口 / 敬語）
   - 得意分野
   - してほしくないこと
3. Discord で「自己紹介して」と話しかけて、性格が出てるか確認

## 📅 定期実行の設定（一番便利）

「朝7時にニュースまとめて」「毎月1日に家計レポート」など、
Cron的な定期タスクが組めます。

- Dashboard の「**Scheduler / Cron**」から設定
- cron式（`0 7 * * *` = 毎日7時）と、実行内容を自然言語で記述
- 例: 「TickTickの今日のタスクをDiscordに送って」

## 🤖 別の LLM プロバイダを試す

> どのプロバイダが「賢い × 無料 × プライバシー安心」かは
> 👉 [プライバシー早見表](./プライバシー早見表.md) を参照！

`.env` に複数のキーを書いて、Dashboard で切り替えられます。

### Cerebras（無料1M tokens/日、爆速）
```bash
# .env に追加（OpenAI互換扱い）
OPENAI_API_KEY=csk-...                    # Cerebrasのキー
OPENAI_BASE_URL=https://api.cerebras.ai/v1
```
※ Dashboard の Provider Settings で base URL も指定する必要あり

### Mistral（エージェント native 対応最強）
```bash
MISTRAL_API_KEY=...
```

### OpenRouter（1キーで100モデル試せる）
```bash
OPENROUTER_API_KEY=sk-or-...
```

## 💚 LINE 連携（HTTPS が必要）

LINE は Webhook が **HTTPS必須** なので、トンネル設定が要ります。

### 方法A: Cloudflare Tunnel（無料、おすすめ）
1. [Cloudflare](https://www.cloudflare.com/) でドメイン取得（無料の `.workers.dev` でもOK）
2. `cloudflared tunnel` でローカル → 公開URLのトンネル作成
3. LINE Developers で Webhook URL に公開URLを設定

### 方法B: ngrok（一番カンタン、URLが変わる）
1. https://ngrok.com/ でアカウント作成
2. `ngrok http 18789` で公開URL発行
3. 起動毎にURLが変わるので、開発・お試し向き

詳細: https://docs.openclaw.ai/channels/line/

## 💾 お引越し / バックアップ

別PCに移したい、SSD 故障に備えたい時。

### バックアップ
```bash
# Mac/Linux
docker run --rm \
  -v openclaw-starter_openclaw_data:/data \
  -v $(pwd):/backup busybox \
  tar czf /backup/buddy-backup-$(date +%Y%m%d).tar.gz /data

# Windows PowerShell
docker run --rm `
  -v openclaw-starter_openclaw_data:/data `
  -v ${PWD}:/backup busybox `
  tar czf /backup/buddy-backup.tar.gz /data
```

### 復元
```bash
docker run --rm \
  -v openclaw-starter_openclaw_data:/data \
  -v $(pwd):/backup busybox \
  tar xzf /backup/buddy-backup-XXXXXXXX.tar.gz -C /
```

`workspace/` フォルダはホスト側にあるので、フォルダごとコピー&ペーストでOK。
