## ③ Discord Bot の作成【推奨・10分】

Discord から自分のバディに話しかけるための準備。

> LINE 派の人へ: LINE 連携は HTTPS Webhook の準備が要るため、講座当日は Discord で動作確認します。LINE は後日チャレンジ枠です🙏

- [ ] https://discord.com/developers/applications にアクセス
- [ ] 右上「**New Application**」 → 名前は「My Buddy」など好きに
- [ ] 左メニュー「**Bot**」 → 「**Reset Token**」 → 「**Yes, do it!**」
- [ ] 表示された **トークンをメモ帳に保存**（一度しか表示されないので注意！）
- [ ] 同じページの「**Privileged Gateway Intents**」で **3つ全部 ON**:
  - PRESENCE INTENT
  - SERVER MEMBERS INTENT
  - MESSAGE CONTENT INTENT
- [ ] 🔒 **Bot を非公開にする（推奨・他人が勝手に招待できないように）**
  - ⚠️ **順序が大事**（逆だと `Cannot have install fields on a private application` エラー）
  - [ ] 左メニュー「**Installation**」 → 「Install Link」を **「None」** → Save Changes
  - [ ] 左メニュー「**Bot**」 → 「**Public Bot**」を **OFF** → Save Changes
- [ ] 左メニュー「**OAuth2**」 → 「**URL Generator**」
  - SCOPES: `bot` にチェック
  - BOT PERMISSIONS: `Send Messages`, `Read Message History`, `Add Reactions`, `Embed Links`, `Attach Files` にチェック
  - 下に出てきた URL をコピー → ブラウザで開いて自分のサーバーに招待

> 💡 自分専用のDiscordサーバーがない人: 左サイドバーの「+」ボタンから「自分用」を選ぶと無料で作れます
